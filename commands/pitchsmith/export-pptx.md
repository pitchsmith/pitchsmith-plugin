description: Export the current deck's HTML slides to a native .pptx PowerPoint file with editable elements
---
Using the /pptx skill - convert the HTML slides in the active deck's slides directory to a .pptx PowerPoint file.
Save the .pptx file to the active deck's output directory.

Important: Do NOT make slides that are just screenshots or full-slide images.
Each slide must have native editable PowerPoint elements — text boxes, shapes, bullet lists, images, etc.
Screenshots should only be used for background images if needed, never as the entire slide content.
